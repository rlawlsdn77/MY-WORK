## 1. 프로젝트 개요

### 1.1 게임 콘셉트
포켓몬스터 하트골드는 Nintendo DS 플랫폼 기반 RPG입니다. 플레이어는 포켓몬 트레이너로서 성도지방을 모험하며 강력한 포켓몬을 포획·육성하고, 8개 체육관의 관장을 격파하여 포켓몬 리그 우승을 달성합니다.

### 1.2 핵심 게임플레이 루프
**탐색 → 전투 및 포획 → 레벨업 및 진화 → 체육관 도전 → 보상 획득 → 다음 지역 해금**

### 1.3 타겟 플레이어
| 항목 | 내용 |
|------|------|
| **연령대** | 전체 이용가 (6세 이상 권장) |
| **플레이어 타입** | 캐주얼 RPG 팬, 포켓몬스터 팬덤 |
| **예상 플레이 시간** | 40~100시간 (메인 스토리 + 엔드 콘텐츠) |

## 2. 목표 시스템

### 2.1 목표 시스템 1: 포켓몬 포획 및 육성 시스템

#### 2.1.1 개요
플레이어가 야생 포켓몬을 포획하고 전투를 통해 경험치를 얻어 포켓몬을 강화하는 **핵심 진행 및 성장 시스템**입니다.

#### 2.1.2 포켓몬 포획 메커니즘

**몬스터볼 종류 및 사양**

| 볼 종류 | 포획 성공률 | 사용 조건 | 가격 |
|--------|-----------|---------|------|
| 몬스터볼 | ×1.0 | 제한 없음 | 200원 |
| 수퍼볼 | ×1.5 | 제한 없음 | 600원 |
| 하이퍼볼 | ×2.0 | 5번째 배지 이상 | 1,200원 |
| 다크볼 | ×3.0 | 밤 시간대 또는 동굴 내부 | 1,000원 |
| 마스터볼 | 100% | 이벤트 전용 (1개 한정) | 비매품 |

**포획 확률 계산식**

```
포획률(%) = (볼 계수) × (체력 감소 보정) × (상태이상 보정) × (난이도 보정)
```
*(※ 체력 감소 보정: 포켓몬의 현재 체력이 낮을수록 포획 확률 배율이 상승함)*

| 보정 요소 | 배율 |
|----------|------|
| 독/화상/마비 상태 | ×1.5 |
| 수면/빙결 상태 | ×2.5 |
| 야생 레벨 ≤ 평균 레벨 | ×1.0 |
| 야생 레벨 = 평균 레벨 +5~10 | ×0.8 |
| 야생 레벨 ≥ 평균 레벨 +15 | ×0.5 |

**포획 전략**
- **약화 전투**: 기술로 체력을 최소한으로 남겨 체력 감소 보정 확률 극대화
- **상태이상 유도**: 마비/수면 기술 사용으로 확률 2~2.5배 상승
- **볼 선택**: 시간/지형에 맞는 최적의 볼 사용

#### 2.1.3 포켓몬 육성 시스템

**경험치 및 레벨업**
- 전투 승리 또는 야생 포켓몬 포획 시 경험치 획득
- 경험치 곡선: 6단계 성장률 (매우 빠름 ~ 매우 느림)
- 최대 레벨: **100**
- **학습장치**: 비보유 포켓몬도 50% 경험치 분배

**진화 메커니즘**

| 진화 유형 | 설명 | 예시 |
|----------|------|------|
| 레벨 진화 | 특정 레벨 도달 시 | 브케인 Lv.14 → 마그케인 |
| 아이템 진화 | 진화석 사용 | 냄새꼬 + 태양의돌 → 아르코 |
| 조건 진화 | 친밀도/시간/장소 | 이브이 + 친밀도 220+ + 낮 → 에브이 |
| 교환 진화 | 플레이어 교환 시 | 윤겔라 교환 → 후딘 |
| 특수 진화 | 특정 기술/아이템 | 스라크 + 금속코트 교환 → 핫삼 |

**개체값(IV) 및 노력치(EV) 시스템**

| 항목 | 범위 | 영향도 | 설명 |
|------|------|--------|------|
| **개체값(IV)** | 0~31 | ~15% | 포획 시 결정되는 선천적 능력치 |
| **노력치(EV)** | 0~252(단일) | ~25% | 전투 승리로 축적되는 후천적 능력치 |
| 총합 EV | 0~510 | - | 모든 능력치 EV의 합계 제한 |

**성격(Nature) 시스템**
- **종류**: 25가지
- **효과**: 1개 능력치 +10%, 1개 능력치 -10% (일부 무보정)
- **결정**: 포획 시 랜덤, 특정 특성으로 유도 가능

#### 2.1.4 친밀도 및 유대감 시스템

**친밀도 수치: 0~255**

| 친밀도 범위 | 효과 |
|-----------|------|
| 0~50 | 기본 상태, 특수 효과 없음 |
| 51~100 | 교환 포켓몬 복종 확률 상승 |
| 101~150 | 은혜갚기 기술 위력 증가 시작 |
| 151~200 | 경험치 보너스 조건 해금 |
| 201~255 | 은혜갚기 최대 위력, 친밀도 진화 조건 충족 |

**친밀도 변화 요소**
- ↑ 상승: 파티 동행, 레벨업, 회복 아이템 사용, 미용실 이용
- ↓ 하락: 전투 중 기절, 쓴맛 약물 사용

#### 2.1.5 포켓몬 팀 구성 권장사항
- 최대 6마리 동시 보유 가능
- **타입 다양성** 확보 권장 (공격/방어/속도 밸런스)
- NPC 가이드를 통한 상성 정보 제공

#### 2.2.3 체육관 전투 메커니즘

**전투 구조**

1. **체육관 보조 트레이너 전투** (선택 사항)
   - 관장 도달 전 경로를 지키는 트레이너들
   - 승리 시: 상금, 경험치, 퍼즐 힌트 획득

2. **관장 전투** (필수)
   - 관장의 정예 포켓몬 파티 (3~6마리)
   - 배지 및 기술머신(TM) 획득

**전투 규칙**

| 규칙 | 내용 |
|------|------|
| **파티 규모** | 플레이어 6마리 vs 관장 3~6마리 |
| **포획 가능** | 불가능 (관장 전투 중) |
| **도망치기** | 불가능 (패배 시 최근 포켓몬 센터로 복귀, 소지금 50% 감소) |
| **필드 환경** | 체육관별 특수 환경 적용 (날씨, 필드 상태 등) |

**관장 AI 난이도 (3단계)**

| 난이도 | 관장 | AI 행동 |
|--------|------|--------|
| **초급** | 1~2번째 | 위력 중심 공격, 회복 1회 이하, 약점 무시 |
| **중급** | 3~5번째 | 상성 고려, 전략적 회복 1~2회, 상태이상 활용 |
| **고급** | 6~8번째 | 최적 전술, 포켓몬 교체 유도, 회복 2~3회, 랭크 조작 |

#### 2.3.3 타입 상성 데미지 시스템

**데미지 배율 정리**

| 상황 | 배율 | 예시 |
|------|------|------|
| 유리한 상성 | ×2.0 | 물 기술 vs 불꽃 포켓몬 |
| 불리한 상성 | ×0.5 | 풀 기술 vs 불꽃 포켓몬 |
| 완전 무효 | ×0.0 | 노말 기술 vs 고스트 포켓몬 |

**복합 타입 상성 계산**

```
최종 데미지 = 기본 데미지 × (타입1 계수) × (타입2 계수)

예: 물/비행 포켓몬에게 전기 기술 공격
= 기본 데미지 × 2.0(물 약점) × 2.0(비행 약점) = 4배 데미지
```

#### 2.3.4 타입별 전투 전략

**공격적 상성 활용 (Offensive)**
- 포켓몬당 4개 기술로 다양한 타입 커버
- 예: 물/땅(전기 약점 무효화) + 풀(전기 저항) 조합으로 약점 보완

**방어적 상성 활용 (Defensive)**
- 타입 저항으로 생존율 증가
- 예: 강철/비행 - 약점 최소화, 무효 타입 보유

**팀 구성 가이드**

| 진행 단계 | 전략 | 예시 |
|----------|------|------|
| **초반** | 3각 상성 형성 | 물 - 풀 - 불꽃 삼각 구도 |
| **중반** | 타입 다양화 | 전기, 비행, 땅 타입 추가 |
| **후반** | 약점 최소화 | 복합 타입 배치를 통한 약점 상쇄 |

#### 2.3.5 특수 전투 콘텐츠

**1️⃣ 타입 챌린지 배틀 (Type Challenge)**

제약 조건
- 플레이어 파티: 동일 단일 타입 6마리
- 상대: 해당 타입의 약점을 노리는 포켓몬

보상
- 타입별 스탯 증강제
- 경험치 ×1.5 보너스
- 희귀 기술머신(TM)

**2️⃣ 상성 극복 배틀 (Weakness Override)**

제약 조건
- 상대 타입: 고정 (예: 불꽃)
- 플레이어: 약점 타입(물, 땅, 바위) **제외** 타입만 가능
- 기술 선택으로만 유리한 상성 창출

보상
- 상성 극복 메달
- 전승 상금 ×3배
- 숨겨진 특성 해금

**3️⃣ 야타가라스 리그 (Type Championship)**

규격
- 월 1회 개최, 타입별 토너먼트
- 단일 타입 6마리 팀으로 8명 트레이너 연전
- 우승 시 타입 메달 + 고개체값 포켓몬 + 전설의 포켓몬 만남 이벤트

#### 2.3.6 타입 상성 교육 시스템

**Play 도감 (Type Dex)**
- 각 타입의 강점/약점/저항 정보 제공
- 전투 중 상성 효과 실시간 피드백 ("효과가 굉장했다!" 등)
- 도감 완성도에 따라 추가 팁 해금

**NPC 타입 전문가 (Type Master)**
- 플레이어 팀 분석 및 타입 다양성 조언
- 타입별 트레이닝 아이템 판매
- 약점 타입 포켓몬 교환 가능

#### 2.3.7 타입 상성 게임플레이 영향

**동적 난이도 조정**

```
타입 다양성 점수 = 중복 최소화 점수 - 편중 페널티 + 밸런스 점수

난이도 조정:
- 타입 다양 우수: 기본 난이도 유지
- 타입 편중 (예: 불꽃 4마리): 난이도 +2단계
  → 상대의 약점 공략 기술 사용 빈도 증가
  → 상대 포켓몬 레벨 상향 조정
```

**AI 전술 시뮬레이션**

상대 AI의 행동 패턴
1. 플레이어 포켓몬 타입 분석
2. 약점 타입을 공략할 수 있는 포켓몬 우선 배치
3. 선제공격 기술 활용도 증가
4. 상성 최적화를 위한 실시간 포켓몬 교체

## 4. 게임 밸런싱

### 4.1 난이도 곡선 및 동적 조정

**기본 원칙**
- 선형적 권장 레벨 가이드 제시
- 중반 자유 탐험 구간(진청/담청/황토마을)에서 **동적 난이도 조정** 적용

**스케일링 메커니즘**
```
진입 시점 파티의 평균 레벨 → 관장 포켓몬 레벨 자동 조정
단, 최저/최고 상한선(Cap)으로 난이도 급변화 방지
```

### 4.2 타입 밸런싱

**18개 타입 간 상성 관계 엄격 적용**
- 2배 데미지, 0.5배 데미지, 무효화 메커니즘
- 목표: 단일 포켓몬 무쌍 플레이 방지, 다채로운 교체 유도

### 4.3 포켓몬 성능 (종족값 BST) 배분

| 포켓몬 분류 | 종족값 범위 | 비고 |
|-----------|----------|------|
| 초전설 | 600~680 | 최상급 성능 |
| 최종 진화/후반 포켓몬 | 450~600 | 강력한 고급 포켓몬 |
| 중급 포켓몬 | 300~450 | 진화 중기 포켓몬 |
| 초급 포켓몬 | 180~350 | 미진화/초반 야생 포켓몬 |

**설계 의도**: 플레이어의 지속적 성장 및 진화 욕구 자극

## 5. 사용자 인터페이스(UI)

### 5.1 듀얼 스크린 활용

**상단 화면 (Main Screen)**
- 필드 이동: 월드 맵 및 플레이어 캐릭터 시점
- 전투: 포켓몬 배틀 애니메이션, HP바, 경험치 게이지, 상태이상 이펙트

**하단 화면 (Touch Screen)**

*필드 이동 중*
- 퀵 메뉴: 런닝슈즈 활성화, 가방, 포켓몬 파티, 포켓기어, 리포트(저장)
- 메뉴 불필요, 항상 터치 가능

*전투 진행 중*
- 4분할 대형 레이아웃: '싸운다' | '포켓몬' | '가방' | '도망친다'
- '싸운다' 선택 시 4개 기술 표시, 직관적 터치 조작

## 6. 보상 및 진행도 시스템

### 6.1 경험치 수급

**기본 경험치**
```
야생 포켓몬 격파 기준 = 1.0배
트레이너 포켓몬 격파 = 1.5배 보너스
```

**추가 보너스**
- 교환받은 포켓몬: 1.5~1.7배 추가 경험치 배율

### 6.2 상금 수급

**계산식**
```
상금 = 트레이너 클래스 기본값 × 포켓몬 레벨
```

**배수 적용**
- 부적금화 아이템 소지: 상금 ×2배
